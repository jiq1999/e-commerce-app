## FEATURES:

- LOOK FOR DIFFERENT PRODUCTS BETWEEN FOUR CATEGORIES OR SEARCH THEM.
- CHANGE THE ORDER AND SORT.
- LOG IN OR REGISTER.
- UPLOAD YOUR OWN PRODUCTS.
- ADD TO CART YOUR FAVORITE PRODUCTS.
- BUY PRODUCTS VIA STRIPE.

## APP DEVELOPED USING:

- HTML
- CSS
- JAVASCRIPT
- REACT
- REDUX
- NODEJS
- EXPRESS
- SEQUELIZE
- POSTGRESQL

## APIs/SDKs

- STRIPE (Payments)
- FIREBASE (Storage)
- AUTH0 (LogIn)
